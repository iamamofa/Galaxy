Introduction to R Programming

Why Learn R?

R is a programming language designed for statistical computing and graphics. It is widely used in bioinformatics, genomics, and data science to analyze, visualize, and model biological data.

  • Open-source and cross-platform
  • Powerful for data manipulation and visualization
  • Strong community and extensive package ecosystem (CRAN, Bioconductor)
  • Ideal for reproducible research and reporting

Installing R and RStudio

Steps

# Install R (Ubuntu/Linux)
$ sudo apt install r-base

# Install RStudio (Optional IDE)
Download from: https://posit.co/download/rstudio/

# Start R
$ R

Basic R Commands

Working with Variables

x <- 10

sum <- x + y

Vectors and Data Frames

# Create a vector
genes <- c("ctxA", "tcpA", "toxR")

Basic Plots

# Create a scatter plot
x <- 1:10
y <- x^2
plot(x, y, main="Simple Scatter Plot", xlab="X Values", ylab="Y = X^2", col="blue")

Example: Analyzing Cholera Data

Sample Script

# Read sample metadata
cholera_data <- read.csv("cholera_samples.csv")

# Summarize
summary(cholera_data)

# Plot coverage distribution
hist(cholera_data$Coverage, main="Coverage Distribution", col="lightblue")

# Filter low-coverage samples
filtered <- subset(cholera_data, Coverage > 30)
print(filtered)
