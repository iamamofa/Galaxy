About Nextstrain

Nextstrain is an open-source project that harnesses pathogen genome data for real-time epidemiological insights. It allows exploration of evolving pathogen populations, providing interactive visualizations for scientists, epidemiologists, and public health officials.

Core Pathogens

  • Cholera: Tracking Vibrio cholerae outbreaks globally
  • SARS-CoV-2: COVID-19 virus analysis and clade tracking
  • Mpox: Evolution and outbreak tracking in DRC
  • H5N1 Influenza: Cattle and avian outbreaks
  • Seasonal Influenza
  • Rabies, Yellow Fever, RSV, Ebola, Chikungunya, HIV, Lassa, Measles, Mumps, Rubella, and more

Philosophy

Pathogens accumulate mutations over time. By reconstructing phylogenies, Nextstrain provides insights into transmission patterns, epidemic growth, and geographic spread.

Open Source Tooling

  • Nextclade: Clade assignment, mutation calling, and sequence quality checks
  • Augur: Bioinformatic workflows for genomic epidemiology
  • Auspice: Visualization of pathogen evolution
  • Nextstrain CLI: Command-line access to pipelines and datasets

Example Command

Run a Nextstrain analysis using the CLI:

nextstrain build --config my_config.yaml --output results/
nextstrain view results/

Community and Contributions

Nextstrain enables community contributions through shared datasets, narratives, and open-source tools. The project is supported by extensive collaborations with global research groups.
