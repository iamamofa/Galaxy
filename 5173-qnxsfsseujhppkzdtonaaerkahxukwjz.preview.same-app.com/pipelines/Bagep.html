BAGEP: Bacterial Genome Pipeline

BAGEP is an automated and scalable workflow for analyzing whole genome sequencing of bacterial samples. It also generates interactive heatmaps for SNP visualization across the genome.

Installation

Clone the BAGEP repository:

git clone https://github.com/idolawoye/BAGEP.git
cd BAGEP

Ensure you have Linux OS (Ubuntu 16.04+ recommended) and Miniconda installed. For Windows, use WSL.

Create the Conda Environment

conda env create -f environment.yml
conda activate bagep

Dependencies

  • Fastp
  • Snippy
  • IQ-TREE
  • Abricate
  • Snakemake
  • Centrifuge
  • R libraries: vcfR, ggplot2, heatmaply

Setup Centrifuge Database

wget -c ftp://ftp.ccb.jhu.edu/pub/infphilo/centrifuge/data/p_compressed+h+v.tar.gz
mkdir $HOME/centrifuge-db
tar -C $HOME/centrifuge-db -zxvf p_compressed+h+v.tar.gz
export CENTRIFUGE_DEFAULT_DB=$HOME/centrifuge-db/p_compressed+h+v

Running BAGEP

Ensure your paired-end FASTQ files are named as SampleName_R1.fastq.gz and SampleName_R2.fastq.gz, and placed in a fastq/ folder. Then run:

snakemake --config ref={your_reference_genome}

Outputs

  • Trimmed FASTQ files in fastp/
  • Taxonomic classification in taxonomy/fastq/ (interactive Krona HTML)
  • AMR detection in amr_output TAB file
  • Core genome and whole genome alignments in results/
  • Phylogenetic tree (*.treefile)
  • SNP heatmap in heatmap_output.html

Citation

Olawoye IB, Frost SDW, Happi CT. 2020. The Bacteria Genome Pipeline (BAGEP): an automated, scalable workflow for bacteria genomes with Snakemake. PeerJ 8:e10121. DOI
