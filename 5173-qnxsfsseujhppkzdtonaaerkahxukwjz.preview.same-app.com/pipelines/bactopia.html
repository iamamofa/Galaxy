Bactopia: Bacterial Genome Analysis Pipeline

Bactopia is a flexible pipeline designed to process bacterial genomes efficiently, allowing quality control, assembly, annotation, sequence typing, and comparative analyses.

Features

  • Per-isolate workflow with Nextflow
  • Quality control, assembly, and annotation
  • Minmer sketch queries and sequence typing
  • Comparative analyses: pan-genome, phylogenetics, summary reports
  • Predictable output structure for easy downstream analyses

Installation

Recommended method using Mamba:

mamba create -y -n bactopia -c conda-forge -c bioconda bactopia
conda activate bactopia
bactopia datasets

Running Bactopia

Examples:

Paired-end reads:
bactopia --R1 R1.fastq.gz --R2 R2.fastq.gz --sample SAMPLE_NAME --datasets datasets/ --outdir OUTDIR

Single-end reads:
bactopia --SE SAMPLE.fastq.gz --sample SAMPLE --datasets datasets/ --outdir OUTDIR

Multiple samples:
bactopia prepare MY-FASTQS/ > fastqs.txt
bactopia --fastqs fastqs.txt --datasets datasets --outdir OUTDIR

Single ENA/SRA experiment:
bactopia --accession SRX000000 --datasets datasets --outdir OUTDIR

Documentation

Full documentation, tutorials, and advanced usage are available at https://bactopia.github.io/.
