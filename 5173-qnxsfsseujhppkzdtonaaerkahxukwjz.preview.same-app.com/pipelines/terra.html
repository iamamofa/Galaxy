Introduction

Terra is a cloud-based platform developed by the Broad Institute, in collaboration with Microsoft and Verily. It enables secure sharing, large-scale biomedical data analysis, and global collaboration.

Key Features

  • Analyze large-scale biomedical data in the cloud
  • Secure sharing of data and workflows
  • Collaborate globally across research teams
  • Support for reproducible analysis workflows
  • Integrates with Microsoft Azure and other cloud resources

Example Workflows

Terra supports reproducible workflows. For example, running a data analysis pipeline:

# Example workflow command in Terra environment
terrapy run workflow.yaml --input samples.csv --output results/

Workflows can be shared with collaborators, and results can be exported securely.
