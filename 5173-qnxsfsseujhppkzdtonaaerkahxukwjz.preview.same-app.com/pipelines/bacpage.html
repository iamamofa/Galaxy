Bacpage: Bacterial Genome Assembly and Analysis Pipeline

Bacpage is an easy-to-use, modular pipeline for bacterial genome assembly and analysis using ONT or Illumina data. It automates quality control, assembly, variant calling, and phylogenetic analysis.

Features

  • Reference-based and de novo genome assembly
  • Illumina and ONT data support
  • Quality control and variant calling
  • Phylogenetic tree inference (IQ-TREE)
  • MLST, AMR, and plasmid detection

Installation Guide

Step 1: Install Mamba (recommended Conda alternative)

curl -L -O "https://github.com/conda-forge/miniforge/releases/latest/download/Miniforge3-$(uname)-$(uname -m).sh"
bash Miniforge3-$(uname)-$(uname -m).sh

Step 2: Clone the Bacpage Repository

git clone https://github.com/CholGen/bacpage.git
cd bacpage/

Step 3: Create and Activate the Environment

mamba env create -f environment.yaml
mamba activate bacpage

Step 4: Install the Bacpage Command

pip install .

Step 5: Test the Installation

bacpage -h
bacpage version

Usage Example

Run a Reference-Based Assembly

mamba activate bacpage
bacpage setup my_project
bacpage assemble my_project

Updating Bacpage

cd bacpage/
mamba activate bacpage
git pull
mamba env update -f environment.yaml
pip install .
