Unix/Linux Introduction

Essential Commands for Bioinformatics

Understanding Unix/Linux command line is fundamental for bioinformatics work. The command line provides powerful tools for data manipulation, file management, and running bioinformatics software.

Why Unix/Linux for Bioinformatics?

  • Most bioinformatics tools are designed for Unix/Linux environments
  • Efficient handling of large genomic datasets
  • Automation through scripting
  • Remote server access for high-performance computing

Basic Navigation Commands

pwd - Print Working Directory

Shows your current location in the file system

$ pwd
/home/user/cholera_analysis

ls - List Directory Contents

Display files and directories

$ ls -lh
drwxr-xr-x 2 user staff 64B Oct 13 08:00 raw_data
drwxr-xr-x 2 user staff 64B Oct 13 08:00 results
-rw-r--r-- 1 user staff 1.5G Oct 13 08:00 SRR24446250_1.fastq.gz

Useful flags:

  • -l: Long format with details
  • -h: Human-readable file sizes
  • -a: Show hidden files
  • -t: Sort by modification time

cd - Change Directory

$ cd raw_data
$ cd .. # Go up one directory
$ cd ~ # Go to home directory
$ cd - # Go to previous directory

File Manipulation Commands

mkdir - Create Directories

$ mkdir -p cholera_project/raw_data
$ mkdir -p cholera_project/results/{assembly,annotation,qc}

cp & mv - Copy and Move Files

$ cp sample.fastq backup/sample.fastq
$ cp -r raw_data/ backup/ # Copy directory recursively
$ mv old_name.txt new_name.txt # Rename

Working with Files

cat, head, tail - View File Contents

$ head -n 10 sample.fastq # First 10 lines
$ tail -n 20 logfile.txt # Last 20 lines
$ cat file1.txt file2.txt > combined.txt # Concatenate files

less & more - Page Through Files

$ less large_file.txt

Navigation in less:

  • Space: Next page
  • b: Previous page
  • /pattern: Search forward
  • q: Quit

grep - Search Text Patterns

$ grep "ATCG" sequences.fasta
$ grep -c "ATCG" sequences.fasta

Bioinformatics-Specific Commands

wc - Count Lines, Words, Characters

Useful for counting sequences in FASTA/FASTQ files

$ wc -l sample.fastq # Count lines
8000000 sample.fastq
# FASTQ has 4 lines per read, so: 8000000 / 4 = 2,000,000 reads

Pipes and Redirection

Chain commands together for powerful data processing

$ grep ">" sequences.fasta | wc -l # Count sequences
$ cat *.fastq > all_reads.fastq # Combine files
$ zcat sample.fastq.gz | head -n 4 # View compressed files

Practical Exercises

Exercise 1: Setting Up Your Workspace

Create a directory structure for a cholera analysis project:

$ mkdir -p cholera_project/{raw_data,qc_reports,trimmed_data,assembly,annotation}
$ cd cholera_project
$ ls -l

Exercise 2: Working with FASTQ Files

Download and inspect a sample FASTQ file:

# View first read in FASTQ file
$ head -n 4 sample.fastq

# Count total number of reads
$ echo $(cat sample.fastq | wc -l)/4 | bc

# Search for a specific sequence pattern
$ grep -B 1 -A 2 "ATCGATCG" sample.fastq

Exercise 3: File Compression

Genomic files are large - learn to compress them:

# Compress a file
$ gzip large_file.fastq

# Decompress
$ gunzip large_file.fastq.gz

# View compressed file without decompressing
$ zcat file.fastq.gz | head

Useful Keyboard Shortcuts

  • Tab: Auto-complete filenames and commands
  • Ctrl + C: Stop running command
  • Ctrl + Z: Suspend current process
  • Ctrl + R: Search command history
  • Ctrl + A: Move to beginning of line
  • Ctrl + E: Move to end of line
  • ↑/↓ arrows: Navigate command history
